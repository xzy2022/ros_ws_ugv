## 测试车轮控制
```
roslaunch my_robot_bringup sim_env.launch
rosrun my_robot_control wheel_control_test.py _robot_name:=smart _mode:=straight _speed:=1.0
rostopic echo /ground_truth/state -n 1 > tmp.txt
```
基本可以确定后轮的旋转速度的确是角速度，因为半径0.3m，发布1.0的后轮速度，/ground_truth/state话题下得到的真实速度为0.29940110522094243
